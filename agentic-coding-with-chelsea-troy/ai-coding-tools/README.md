# Overview

This directory contains plans for the AI coding tools project. Each plan is a detailed outline of the steps and processes involved in implementing specific features or functionalities within the project. These plans serve as a roadmap for development, ensuring that all team members are aligned and have a clear understanding of the tasks at hand. It also serves as cross reference for different agentic ai coding tools such as claude code, github copilot, etc...

## Scripts

Determinisitic code for running repeatable tasks.

## Plans

Document detailing intended feature executions.

## Docs

Documents explaining repo structure and function.

## Process

Markdown files discribing workflows like refactoring.

# MCP

Create custom MCP server for the different agent command, in such case the commands are shareable across different agentic ai coding tools such as claude code, github copilot, etc... and can be used by any of them without the need to create a new server for each tool. Scope the mcp server to related tool in order to avoid fludding the context window.

```
claude mcp list
```
