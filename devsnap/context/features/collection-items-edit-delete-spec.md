# Collection Edit/Delete Collection

## Overview

Add functionality to edit, delete and add to favorite collections.

## Requirements

- Add buttons on /collections/[id] to edit, delete and favorite. Do not implement favorites yet, just the icon/button.
- Add a modal for edit to edit the meta data. Add a confirmation on delete. Items should NOT be deleted, they just will not exist in that collection anymore.
- On the cards at /collections and dashboard, have the 3 dots icon show a dropdown with edit, delete and favorite. Clicking anywhere else in the card, will go to that collection page.
