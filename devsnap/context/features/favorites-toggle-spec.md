# Favorite Toggle

## Overview

Ability to favorite/unfavorite items and collections directly from the item/collection list views and detail pages.

## Requirements

- Add star icon button to item and collection list rows and detail pages
- Clicking star toggles favorite status for that item/collection
- Update UI immediately to reflect new favorite status (filled star for favorited, outlined for not)
- Persist favorite status to backend
- Update favorite count in TopBar if applicable
