# Start Action

1. Read current-feature.md - verify Goals are populated
2. If empty, error: "Run /feature load first"
3. Set Status to "In Progress"
4. Git folder is up one directory above the project root, so you can run git commands as normal. Run `git checkout -b [feature/[feature-name]]` to create new branch.
5. Create and checkout the feature branch (derive name from H1 heading)
6. List the goals, then implement them one by one
